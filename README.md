# Worknoon Chat Backend

Node.js + Express + TypeScript real-time gateway with MongoDB persistence and Socket.io.

## Quick start

```bash
cp .env.example .env
npm install
npm run dev
```

## Scripts

| Script | Purpose |
|--------|---------|
| `npm run dev` | Watch mode via `tsx` |
| `npm run build` | Emit `dist/` |
| `npm run start` | Run production build |
| `npm run typecheck` | Strict TS validation |
